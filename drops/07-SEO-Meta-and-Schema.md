# Groundwork Solutions: SEO Meta and Schema Templates

Every meta tag, OG tag, Twitter card, and JSON-LD block needed for the site. Patterns first, then a full table of titles and metas for all 22 pages.

---

## 1. Title tag rules

- 50 to 60 characters ideal, 65 max (Google truncates around 60 to 65)
- Primary keyword first, brand last
- Use pipe `|` as separator, no em-dash
- Include a city or modifier when natural
- Sentence case for the phrase, brand always "Groundwork"

**Formulas by page type:**

| Page type | Formula |
|---|---|
| Home | `[Primary Keyword] [City] \| [What we do] \| Groundwork` |
| Service detail | `[Service] [City] \| [Differentiator or Variant] \| Groundwork` |
| Industry detail | `[Service Type] for [Industry] \| [City] \| Groundwork` |
| Overview pages | `[Topic] \| [Subhead] \| Groundwork` |
| About | `About Groundwork \| [Differentiator] \| [City]` |
| Contact | `Contact Groundwork \| [Action] \| [Phone]` |

---

## 2. Meta description rules

- 150 to 160 characters ideal
- Include primary keyword and city
- Say what the service is and who it's for
- Add a real differentiator (24 hour quotes, owner-operated, tight-lot capable, etc.)
- End with a soft CTA or phone number
- No banned words from Content Style Guide

**Pattern:** `[Service or What we do] in [City]. [Specifics or scope]. [Differentiator]. [CTA or phone].`

**Example (Excavation):** `Excavation in Sacramento for foundations, basements, and footings. Mini excavator and skid steer fleet, tight-lot capable. Quotes within 24 hours. Call 530-559-8502.`

### 5.6 FAQPage schema (any page with an FAQ section)

```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "[Question text]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "[Answer text, plain prose, 40 to 90 words]"
      }
    },
    {
      "@type": "Question",
      "name": "[Question 2]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "[Answer 2]"
      }
    }
  ]
}
</script>
```

Rules:
- Question text in the schema must match the visible question text on the page exactly
- Answer text must match the visible answer text exactly
- Don't include FAQ schema if there's no visible FAQ on the page (cloaking penalty)

## 7. OG image specs

One default OG image plus per-page where it adds value.

**Default:** `og-default.jpg`, 1200x630, brand colors, logo lockup, text reads "Site Prep, Northern California".

**Service-specific OG (optional):** real photo of that service in action, with a small logo lockup overlay bottom-right.

**Industry-specific OG (optional):** photo + headline overlay like "Site Prep for Pool Builders".

If you don't have time to make per-page OG images, use the default for everything.

**File specs:**
- 1200x630 px exact
- JPG, quality 85
- Under 100 KB
- Hosted in GHL Media or your own subdomain
- Use absolute URLs in OG meta tags

## 8. Robots and sitemap

Allow: /
Disallow: /thank-you/
Disallow: /admin/
